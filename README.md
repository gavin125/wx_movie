# 微信小程序版豆瓣电影

## 概述
微信小程序版豆瓣电影（非官方出品）
- 数据来源：[豆瓣电影API](https://developers.douban.com/wiki/?title=movie_v2)
- 开发工具：微信开发者工具 0.14.140900


## 功能
- 电影榜单列表
- 电影搜索
- 电影条目信息
- 影人条目信息
